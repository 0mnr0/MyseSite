Движок не требующий кода

Myse представляет собой два инструмента: MyseEngine и MyseWriter

- MyseEngine представляет собой заранее заготовленый движок который готовый к работе. Для того чтобы начать игру на всех доступных платформах - нажмите кнопку в MyseWriter
- MyseWriter представляет собой программу которая позволяет писать саму новеллу. Выбирайте опции для своей истории и смотрите результат на любой платформе

Играть как угодно

:Myse может работать сразу с тремя платформами

Смартфон является оптимальным вариантом. Но Myse умеет переносить игровые данные на следующие платформы
Смарт-часы - самый простой процесс игры. Там не будет лишних деталей и будут отключены мало-важные функции игры
ПК или Web (создание не начато). Возможно, будет иметь самый "приятный" игровой процесс за счёт анимаций

Открытый исходный код

Хотите добавить уникальную фишку в вашу игру?

У Myse всегда будет Открытый код, поэтому с безопасностью и идеями никогда не возникнет проблем
Вы всегда можете узнать как работает движок или поменять его если хотите сделать игру уникальной
Описание движка
Myse - игровой движок который создан примущественно для платформы Android. Он позволяет с помощью всего одной программы для ПК создавать игры жанра визуальные новеллы
Без кода!
Myse не требует кода что уже делает его простым движком для разработки своей игры.
Для обеспечения простой и понятной реализации написании игры, был выбран подход «простоты».
Есть специальная программа в которой всё заранее подготовлено, автору игры нужно просто выбрать действие, например,
вывести текст за главного героя, написать сам текст, выбрать изображение героя и сохранить действие.
Востребованность
Перед реализацией я ознакомился с существующими реализациями подобного проекта в той же среде, и должен сказать, движков для реализации новелл не так уж и много:

RenPy - бесплатный, свободный и открытый движок для создания визуальных романов (графических квестов с диалоговой системой) в 2D-графике. Самый популярный пример – бесконечное лето

Unity –бесплатный, свободный и открытый движок для создания любых типов игр, включая новеллы. Примеров много, один из них - Along the Edge
Игра где угодно!
Что делает движок еще более интересным, так это мультиплатформенность.

Игра может быть рядом где угодно, игровой процесс будет продолжаться на запястье без потери прогресса, потому что игра работает на Смарт-часах.
Игровой процесс на часах можно настроить! Все настройки для смарт-часов устанавливаются с телефона. Конечно, не так гибко как на телефоне, но всё же можно :)
Так-же Myse поможет:
Сделать простые новеллы: Авторы смогут создавать свои игры без знаний в коде, распространять свою игру где и как угодно, обновлять и модифицировать её
Так-же Myse поможет:
Ввести новичков в создание игр: Тот кто ни разу не делал игры может понять как примерно работают движки, хоть Myse и не даст возможность писать свой код, но для создания хорошей новеллы его хватит.
Так-же Myse поможет:
Просто создать новеллу: Myse может подойти тем людям которым больше и не нужно – не надо кодить, не надо собирать проект, написал сценарий – игра готова!
